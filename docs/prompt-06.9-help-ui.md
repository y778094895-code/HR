# Help & Support UI Documentation

## Overview
The Help Center is a self-service portal designed to reduce support overhead by providing easy access to documentation and a streamlined ticketing workflow.

## Features

### 1. Knowledge Base
-   **Search**: Real-time filtering of help articles.
-   **Categories**: Organized grouping of topics (Getting Started, Troubleshooting, etc.).
-   **Articles**: Detailed guides (mock content).

### 2. Ticket Management
-   **My Tickets**: Table view of the user's submissions with status indicators (Open, In Progress, Resolved).
-   **Create Ticket**: `SideDrawer` form with priority selection and validation.

### 3. Contact Support
-   **Information**: Displays email and phone channels with availability hours.

## Components Used
-   `TabsContainer`: Navigation.
-   `DataTable`: Ticket list.
-   `SideDrawer`: New ticket form.
-   `Lucide Icons`: Visual indicators for categories and actions.
