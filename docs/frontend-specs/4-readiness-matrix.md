# 4. مصفوفة جاهزية الوظائف (Functionality Readiness Matrix)

### ملاحظات إضافية
1. كل أزرار تحميل التقارير (Export PDF/CSV/Excel) الموجودة بكثرة في صفحات مثل العدالة/التدريب/الأداء/الاستقالات/الأثر تعمل بوضع **Mock Download** محلي.
2. استخدام الإشعارات (Toast system) متصل بالعمليات الوظيفية محلياً أو عبر API.
3. التصديرات لا تمثل بيانات حقيقية من واجهات برمجية لـ Excel/PDF حتى يتم ربطها.
